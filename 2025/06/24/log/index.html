中脑组织额外分析

分析流程：

🔹 Step 1. 数据获取与准备

  • 目标：获取质量可控、脑区明确、AD/Control 明确的 DNA 甲基化原始数据。
  • 来源：
    • GEO（如 GSE153712, GSE109887 等）
    • NIAGADS（如 Knight-ADRC EPIC 数据）
  • 文件类型：
    • .idat（用于重头分析）
    • .csv（标准化 beta 矩阵）
    • phenotype 信息（包含诊断状态、年龄、性别、脑区、RIN、PMI）

🔹 Step 2. 数据预处理与质控（QC）

  • 目标：清洗数据，确保甲基化信号真实、无批次偏差。
  • 流程：
    1. 用 minfi 或 ChAMP 读入 .idat 文件；
    2. 执行 Noob / Functional Normalization；
    3. 删除：
      • 探针检测 p 值 > 0.01；
      • 性染色体探针（如非目标）；
      • 交叉反应探针；
      • 含 SNP 的探针；
    4. 使用 ComBat 或 removeBatchEffect 进行批次校正；
    5. 可估算细胞组成（estimateCellCounts() 或 EpiDISH）用于后续建模控制。

🔹 Step 3. DMP / DMR 分析

  • 目标：找出 AD 和正常人之间存在显著差异甲基化的位点和区域。
  • 方法：
    • limma: 用于 DMP 分析
    • DMRcate / bumphunter: 用于识别 DMR（区域性甲基化改变）
  • 输出：
    • CpG ID、Δβ 值、adj. p-value
    • DMR 起止位置、包含 CpG 数、区域注释

🔹 Step 4. DMR 注释与富集分析

  • 目标：将差异甲基化区域注释到功能基因并解析其生物意义。
  • 方法：
    • 使用 annotatr 或 Illumina EPIC 注释包 注释 DMR 到基因启动子/体区；
    • 用 missMethyl::gometh() 或 clusterProfiler 进行 GO / KEGG 分析；
  • 结果：
    • 与神经功能、炎症、细胞骨架等相关的富集条目；
    • 重点通路如 amyloid-beta clearance、tau phosphorylation、synaptic signaling。

🔹 Step 5. 多组学整合分析（Multi-omics Integration）

这是本流程的核心拓展部分，具体如下：

🔸 5.1 DNA甲基化 × 基因表达（整合分析）

  • 目标：探索 DMR 是否影响其附近基因的表达，发现潜在调控机制。

  • 输入：

    • 同一批次样本或配对数据的 RNA-seq 表达矩阵；
    • DMR 注释到基因列表；

  • 方法：

    • 直接相关性分析：

    • 差异表达整合：
      使用 DESeq2/edgeR 鉴定 DEG，并与 DMR-overlapped genes 取交集；

    • 共表达网络整合：
      使用 WGCNA 模块共建网络，观察模块中 DMR 基因富集情况；

  • 产出：

    • 一批表达受甲基化调控的候选调控基因；
    • 可以构建“DMR → DEG”路径图，建议用 Cytoscape 展示。

🔸 5.2 DNA甲基化 × GWAS（meQTL 分析）

  • 目标：探测 GWAS 风险位点是否通过调控 CpG 位点甲基化水平参与 AD 机制。

  • 输入：

    • AD 相关 GWAS summary statistics（如 IGAP, ADNI）；
    • CpG β值矩阵；

  • 方法：

    • 构建 cis-meQTL 分析模型：

    • 推荐工具：MatrixEQTL, FastQTL

  • 输出：

    • SNP–CpG 对；
    • 可视化 meQTL 热图或 LocusZoom 风格图表；
    • 标记出“GWAS风险位点 → 甲基化改变 → 表达改变”的三级机制。

🔸 5.3 DNA甲基化 × 蛋白质组（如有）

  • 目标：判断 DMR 是否最终影响蛋白表达层。
  • 方法：
    • 构建 CpG–Gene–Protein 三层映射网络；
    • 使用 mixOmics::DIABLO 进行多组学 block-wise 模型整合；
  • 结果：
    • 多组学 signature；
    • 可用于生物标志物筛选或亚型识别。

🔹 Step 6. 网络构建与关键因子识别

  • 目标：将关键 CpG、基因、蛋白组织成调控网络，筛选重要节点（hub genes）。
  • 工具：
    • Cytoscape：可视化
    • igraph / STRING：网络拓扑指标分析（degree, betweenness）
  • 补充：也可结合 OmicsNet 平台快速绘图

🔹 Step 7. 生物学解释与验证建议

  • 结合现有文献确认关键 CpG 所在基因是否已与 AD 相关；
  • 若为新基因，可设计实验验证（qPCR、bisulfite PCR）；
  • 强调 top CpG 或 DMR 的调控机制、脑区特异性和潜在靶向性。
